Documentation

Extract the files after downloading, and run the executable. This should open up a screen similar to the one below.

Here you will have to provide a password that will be needed to start the automation. This password will be essential to safeguard your privacy.

After setting the password your password will be added successfully and you will be redirected towards the automation setup screen

In this window, you will find two options browser automation and desktop automation.

You are able to choose anyone of the two options. For example if you choose browser automation.

On choosing the browser automation one is able to add the different URLs that you want to open when you run the executable file. As an instance we have chosen fiverr.com and other websites.

After choosing the browser urls you can bind it into another file with a .exe extension

Alternatively if can choose the desktop automation option.

Here you are able to provide the location of the application's executable file. You can find this location by typing the name of the application in the start menu and then copying the path of the same.

After adding the paths you can bind it into another file with a .exe extension
